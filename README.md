## Installation:

### Clone the repository: 
 ```bash
   git clone https://github.com/MartineMidttveit/social-media-client
```

### Install dependencies:
 ```bash
   npm install
   ```

### Run the application:
 ```bash
   npm run start
   ```

### Run tests:

 ```bash
   npm run test
   ```
